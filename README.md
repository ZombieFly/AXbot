# 什么是AXbot？
AXbot是一个基于Nonebot开发的机器人，故在使用本机器人之前，你需要先安装并配置好Nonebot，及其前置配置。
# 模块功能
## mws.py:
  <中文Minecraft Wiki搜索引擎模块><br>
 	采用双源搜索：<br>
### 源站点：
minecraft-zh.gamepedia.com<br>
      使用语法：``wiki [搜索内容]``<br>
      实例：``wiki 末影龙``<br>
### bili镜像：
wiki.biligame.com/mc
使用语法：``gpwiki [搜索内容]``<br>
      举例：``gpwiki 末影龙``<br>

## moe.py:
  <萌娘百科搜索引擎模块><br>
  站点：zh.moegirl.org<br>
    使用语法：``moe [搜索内容]``<br>
    举例：``moe 初音ミク``<br>
